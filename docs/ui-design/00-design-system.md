# 本体平台 UI 设计系统规范

> 本文档为所有页面设计的统一参考基准，团队并行开发时必须严格遵循。

---

## 1. 设计原则

| 原则 | 说明 |
|------|------|
| 一致性 | 相同功能使用相同组件和交互模式 |
| 信息层次 | 通过字号、颜色、间距建立清晰的视觉层次 |
| 效率优先 | 减少不必要的点击，核心操作一步触达 |
| 响应式 | 适配 1920×1080 / 1440×900 / 1280×720 三档主流分辨率 |

## 7. 全局布局

### 7.1 应用壳体

```
+----------------------------------------------------------+
| AppSidebar (240px)  |  AppMain                           |
|                     |  +-------------------------------+  |
|  Logo区 (56px高)     |  | AppTopbar (56px)              |  |
|                     |  +-------------------------------+  |
|  导航菜单            |  |                               |  |
|  (分组+图标+文字)    |  |  Page Content                 |  |
|                     |  |  (padding: 24px 32px)          |  |
|                     |  |                               |  |
|  底部折叠按钮        |  |                               |  |
+----------------------------------------------------------+
```

### 7.2 响应式断点

| 断点 | 宽度 | 侧边栏行为 | 内容区padding |
|------|------|------------|---------------|
| Desktop L | ≥1920px | 展开 240px | 32px |
| Desktop M | 1440-1919px | 展开 240px | 24px |
| Desktop S | 1280-1439px | 折叠为 64px 图标模式 | 20px |
| Tablet | <1280px | 抽屉模式(浮层) | 16px |

### 7.3 侧边栏规范

- 宽度：展开 240px / 折叠 64px
- 背景：`--sidebar-bg: #f7f8fa`
- 菜单项高度：40px
- 菜单项左内边距：展开 20px / 折叠居中
- 活跃态：左侧 3px 蓝色竖条 + 背景 `#e8ecf4`
- hover：背景 `#eef0f4`
- 分组标题：12px / 大写 / `--neutral-500`
- 图标尺寸：20×20px

### 7.4 顶栏规范

- 高度：56px
- 背景：`#ffffff`
- 底部边框：`1px solid --neutral-200`
- 左侧：面包屑导航（14px，分隔符 /）
- 右侧：通知铃铛 + 用户头像+姓名

## 8. 通用组件规范

### 8.1 按钮

| 类型 | 背景 | 文字 | 边框 | 高度 | 圆角 |
|------|------|------|------|------|------|
| Primary | `--semantic-500` | #fff | 无 | 36px | 4px |
| Secondary | #fff | `--semantic-600` | `--semantic-200` | 36px | 4px |
| Ghost | transparent | `--neutral-700` | 无 | 36px | 4px |
| Danger | #fa5252 | #fff | 无 | 36px | 4px |
| 小尺寸 | - | - | - | 28px | 4px |

- 字号：14px / 字重：500
- 内边距：0 16px（标准）/ 0 12px（小）
- 禁用态：opacity 0.5 + cursor not-allowed
- hover：背景色加深 10%

### 8.2 输入框

- 高度：36px
- 内边距：8px 12px
- 边框：1px solid `--neutral-200`
- 圆角：6px
- 聚焦：border-color `--semantic-500` + shadow `0 0 0 2px --semantic-50`
- 占位符色：`--neutral-500`
- 字号：14px

### 8.3 表格

- 表头：背景 `--neutral-50`，字号 12px，字重 600，颜色 `--neutral-500`
- 行高：48px
- 行分割：1px solid `--neutral-100`
- hover行：背景 `--neutral-50`
- 选中行：背景 `--semantic-50`
- 分页器：右对齐，14px

### 8.4 卡片

- 背景：`#ffffff`
- 边框：1px solid `--neutral-200`
- 圆角：8px
- 内边距：16px（紧凑）/ 20px（标准）/ 24px（宽松）
- 阴影：sm 级别
- hover 提升：translateY(-1px) + shadow md

### 8.5 标签/徽章

- 高度：22px
- 内边距：0 8px
- 圆角：4px
- 字号：12px
- 颜色搭配：使用对应状态色的浅底+深字

### 8.6 模态框

- 最大宽度：520px（标准）/ 720px（大）/ 960px（超大）
- 圆角：12px
- 内边距：24px
- 遮罩：rgba(0,0,0,0.45)
- 关闭按钮：右上角 24×24
- 标题：18px / 600

### 8.7 抽屉

- 宽度：360px（窄）/ 480px（标准）/ 640px（宽）
- 从右侧滑入，过渡时间 300ms
- 头部：标题 + 关闭按钮，底部 1px 分割线
- 底部操作栏：固定定位，border-top

---

## 9. 图标规范

- 图标库：自定义 SVG（内联）+ @ant-design/icons-vue
- 标准尺寸：16px（行内）/ 20px（菜单）/ 24px（操作）/ 32px（空状态）
- 颜色：继承当前文字颜色（currentColor）
- 交互图标 hover：颜色变为 `--semantic-500`

---

## 10. 动效规范

| 场景 | 时长 | 缓动函数 | 说明 |
|------|------|----------|------|
| 按钮hover | 150ms | ease | 背景色过渡 |
| 面板展开 | 250ms | ease-out | 侧抽屉/面板 |
| 模态框 | 200ms | ease-out | 淡入+缩放 |
| 页面切换 | 300ms | ease | 淡入淡出 |
| 微交互 | 150ms | ease | 图标旋转等 |

---

## 11. 空状态规范

- 居中显示
- 图标/插图：64×64 或 120×120
- 标题：16px / 600 / `--neutral-700`
- 描述：14px / 400 / `--neutral-500`
- 操作按钮：在描述下方 16px

## 13. 无障碍要求

- 文字对比度：≥ 4.5:1（正文）/ ≥ 3:1（大文字）
- 可交互元素：最小点击区域 36×36px
- 焦点态：2px outline + 2px offset
- 键盘导航：Tab 顺序逻辑清晰
- 图标按钮：必须有 aria-label
