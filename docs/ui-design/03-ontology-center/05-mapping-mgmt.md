# 映射管理

> 本体中心 · 本体对象到物理数据源的字段映射

---

## 1. 页面概述

| 维度 | 说明 |
|------|------|
| 页面定位 | 管理本体对象属性与物理数据表字段之间的映射关系，通过可视化连线直观展示绑定状态，支持拖拽建立映射 |
| 用户角色 | 数据架构师、本体工程师、ETL工程师 |
| 核心价值 | 将抽象本体模型与真实数据打通，是本体"落地"为可查询知识图谱的关键桥梁 |

---

| 编号 | 功能点 | 优先级 | 交互类型 | 说明 |
|------|--------|--------|----------|------|
| F01 | 对象选择 | P0 | 下拉选择 | 选择要配置映射的本体对象 |
| F02 | 数据资产绑定 | P0 | 选择器弹窗 | 为对象绑定主数据源/增强数据源 |

| F06 | 映射验证 | P1 | 按钮触发 | 验证映射配置的正确性，取样预览 |

## 4. 各区域详细设计

### 4.1 操作栏

| 属性 | 值 |
|------|-----|
| 高度 | 56px |
| 背景 | `--neutral-0` |
| 下边框 | 1px solid `--neutral-200` |
| 内边距 | 0 24px |
| 布局 | flex，align-items center，justify-content space-between |

左侧元素：

| 元素 | 规格 |
|------|------|
| 对象选择器 | `a-select`，宽 240px，显示 TierBadge + 对象名称 |
| 统计文字 | 14px / 500 / `--neutral-700`，"已映射 8/12" |
| 覆盖率进度 | `a-progress` 线性，宽 120px，颜色根据比例变化 |

右侧按钮组：

| 按钮 | 类型 | 说明 |
|------|------|------|
| 自动映射 | Secondary | 按名称匹配自动映射 |
| 验证 | Secondary | 取样验证映射是否正确 |
| 保存 | Primary | 保存当前映射配置 |

覆盖率颜色规则：
- 0-30%：`--status-error` 红色
- 31-70%：`--kinetic-500` 橙色
- 71-100%：`--dynamic-500` 绿色

### 4.2 左栏 - 本体属性列表

| 属性 | 值 |
|------|-----|
| 宽度 | 280px |
| 背景 | `--neutral-0` |
| 右边框 | 1px solid `--neutral-200` |
| 内边距 | 16px |
| 头部 | "本体属性" 14px/600 + 属性计数 |

属性行：

| 属性 | 值 |
|------|-----|
| 高度 | 40px |
| 布局 | flex，align-items center |
| 左侧圆点 | 8px 圆，已映射 `--dynamic-500`，未映射 `--neutral-300` |
| 属性名 | 14px / 500 / `--neutral-900` |
| 类型标签 | 右侧，pill 12px，如 "string" "number" |
| 拖拽手柄 | 右端 grip 图标，cursor: grab |
| hover | 背景 `--neutral-50` |
| 拖拽中 | 背景 `--semantic-50`，边框虚线 `--semantic-500` |

### 4.3 中栏 - 连线画布

| 属性 | 值 |
|------|-----|
| 宽度 | flex: 1 (最小 200px) |
| 背景 | `--neutral-50`，微点阵图案（dot grid） |
| 实现 | SVG 绘制贝塞尔曲线连线 |

连线样式：

| 状态 | 样式 |
|------|------|
| 正常 | stroke: `--semantic-500`，stroke-width: 2px，opacity 0.7 |
| hover | stroke-width: 3px，opacity 1，显示删除按钮 |
| 拖拽中 | stroke: `--kinetic-500`，stroke-dasharray: 5,5，动画 |
| 类型不匹配 | stroke: `--status-warning`，虚线 |

连线上的操作：
- hover 连线中点出现 ✕ 按钮可删除
- hover 显示 tooltip："属性A → 字段1 (string→varchar)"

## 5. 数据展示规范

| 场景 | 处理方式 |
|------|----------|
| 数字格式 | 属性数/字段数直接整数，覆盖率保留整数百分比 |
| 空状态 - 未选对象 | 全区域居中："请先选择要映射的本体对象" |
| 空状态 - 未绑定 | 中央区域："该对象尚未绑定数据资产" + "绑定数据资产" 按钮 |
| 加载态 | 左右栏各自 Skeleton 列表，连线区域 Spin |
| 错误态 | 验证失败行标红 + tooltip 提示原因 |
| 类型不匹配 | 连线变为橙色虚线 + 警告图标 |

## 6. 响应式适配

| 断点 | 布局变化 |
|------|----------|
| ≥1920px | 三栏正常：280px + flex + 280px，连线区宽裕 |
| 1440-1919px | 三栏：240px + flex + 240px |
| 1280-1439px | 三栏：220px + flex + 220px，字段名可能截断 |
| <1280px | 改为上下布局：左栏和右栏并排，连线区在下方变为表格模式（列表替代可视化连线） |
