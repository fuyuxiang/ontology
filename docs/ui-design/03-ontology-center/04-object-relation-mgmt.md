# 对象与关系管理

> 本体中心 · 本体对象类型与关系的 CRUD 管理

---

## 1. 页面概述

| 维度 | 说明 |
|------|------|
| 页面定位 | 本体核心管理界面，以三栏布局呈现对象类型的层级结构、列表浏览和详细信息，支持完整的增删改查操作 |
| 用户角色 | 本体工程师、数据架构师、业务分析师 |
| 核心价值 | 提供对本体 TBox 层的全面管理能力，直观展示 Tier 分层结构，高效维护对象属性和关系定义 |

---

## 4. 各区域详细设计

### 4.1 左栏 - Tier 摘要面板

| 属性 | 值 |
|------|-----|
| 宽度 | 280px 固定 |
| 背景 | `--neutral-0` |
| 右边框 | 1px solid `--neutral-200` |
| 内边距 | 20px |
| 溢出 | overflow-y: auto |

内部结构：

| 元素 | 规格 |
|------|------|
| 眉批文字 | `--text-caption` 12px / 500 / `--neutral-500`，大写 |
| 区域标题 | `--text-h2` 18px / 600 / `--neutral-900` |
| 副标题 | 14px / 400 / `--neutral-500` |
| 新建按钮 | Secondary Button + 加号图标，位于标题右侧 |
| "全部对象"按钮 | 宽 100%，高 44px，flex 两端对齐，左:标签 右:数量 |
| Tier卡片 | 宽 100%，高 60px，左侧 TierBadge + 文字，右侧数量 |
| 选中态 | 背景 `--semantic-50`，左边框 3px solid 对应Tier颜色 |

Tier颜色编码：
- T1 核心层：`#2E5BFF` 蓝色
- T2 扩展层：`#7950f2` 紫色
- T3 应用层：`#20c997` 绿色

### 4.2 中栏 - 实体列表

| 属性 | 值 |
|------|-----|
| 宽度 | 340px 固定 |
| 背景 | `--neutral-0` |
| 右边框 | 1px solid `--neutral-200` |
| 溢出 | overflow-y: auto |

内部结构：

| 元素 | 规格 |
|------|------|
| 头部眉批 | 同左栏格式 |
| 标题 | "T1 核心对象"（随筛选变化）/ 18px / 600 |
| 计数 | "12 / 35" 格式，14px / `--neutral-500` |
| 搜索框 | 宽 100%，高 36px，左侧搜索图标，圆角 4px |
| EntityCard | 见下方定义 |

EntityCard 组件：

| 属性 | 值 |
|------|-----|
| 高度 | auto（约 72px） |
| 内边距 | 12px 16px |
| 左边框 | 3px solid Tier颜色（选中时显示） |
| 背景 | 默认 transparent，hover `--neutral-50`，选中 `--semantic-50` |
| 左侧 | TierBadge (24×24px 圆) + 对象名称(14px/600) + 英文名(12px/`--neutral-500`) |
| 右侧 | 实例数量 badge（12px，pill 样式） |
| 间距 | 卡片间 margin-bottom 2px |

### 4.3 右栏 - 详情面板

| 属性 | 值 |
|------|-----|
| 宽度 | flex: 1 |
| 背景 | `--neutral-0` |
| 内边距 | 24px |
| 溢出 | overflow-y: auto |

详情头部：

| 元素 | 规格 |
|------|------|
| TierBadge | 32×32px 圆，Tier颜色背景 |
| 对象名称 | `--text-h1` 22px / 600 |
| 层级标签 | "Tier 1 核心层"，12px / `--neutral-500` |
| 编辑按钮 | Secondary Small，"编辑"，右对齐 |

统计卡片区：

| 元素 | 规格 |
|------|------|
| 布局 | grid 4列，gap 12px |
| 卡片 | 圆角 8px，padding 12px，背景 `--neutral-50` |
| 数值 | 20px / 700 / `--neutral-900` |
| 标签 | 12px / 400 / `--neutral-500` |

Tab栏：

| 属性 | 值 |
|------|-----|
| Tab项 | 属性、关系、规则、动作、绑定 |
| 字号 | 14px / 500 |
| 选中态 | 文字 `--semantic-600` + 底部 2px 蓝色线 |
| 间距 | gap 24px，margin-top 20px |

属性Tab表格：

| 列 | 宽度 | 内容 |
|----|------|------|
| 属性名 | 30% | 14px/600 + 英文名12px |
| 类型 | 15% | Badge样式（string/number/boolean等） |
| 必填 | 10% | 开关图标 |
| 描述 | 35% | 14px/400 文字截断 |
| 操作 | 10% | 编辑/删除图标按钮 |

关系Tab列表：

| 元素 | 规格 |
|------|------|
| 行高 | 48px |
| 布局 | 关系名 → 目标对象 (基数)，箭头图标连接 |
| 基数标签 | pill 样式 12px，如 "1:N" "N:M" |
| 类型标签 | 蓝色/橙色/紫色区分 ObjectProperty/association 等 |

## 5. 数据展示规范

| 场景 | 处理方式 |
|------|----------|
| 数字格式 | 实例数量 ≥1000 使用千分位（如 1,234）；≥10000 使用 k 缩写（如 12.3k） |
| 空状态 - 无对象 | 三栏中间+右侧合并显示空状态："暂无对象类型" + "前往构建" 按钮 |
| 空状态 - 未选中 | 右栏显示引导文字："选择左侧对象查看详情" |
| 加载态 | 左栏数字使用 Spin，中栏卡片使用 Skeleton 列表，右栏全区 Spin |
| 错误态 | 接口错误时顶部 Alert 横幅 + 重试 |
| 搜索无结果 | 中栏显示"未找到匹配对象" + 清除搜索链接 |

## 6. 响应式适配

| 断点 | 布局变化 |
|------|----------|
| ≥1920px | 三栏正常展示：280px + 340px + 1fr |
| 1440-1919px | 三栏：280px + 320px + 1fr |
| 1280-1439px | 三栏缩减：248px + 320px + 1fr（参照现有代码） |
| <1280px | 单列堆叠：左栏横向Tier卡片 → 中栏列表 → 右栏详情 |
