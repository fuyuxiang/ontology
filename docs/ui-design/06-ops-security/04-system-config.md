# 系统配置（System Config）

> 运维与安全中心 · 平台全局参数与集成配置管理

---

## 1. 页面概述

| 维度 | 说明 |
|------|------|
| 页面定位 | 平台级系统参数统一配置入口，分组管理基础、认证、存储、AI、通知等配置项 |
| 目标用户 | 系统管理员 |
| 核心价值 | 集中管理平台运行参数，降低配置分散导致的维护成本，支持配置变更追溯 |
| 路由路径 | `/ops/system-config` |
| 所属分组 | 运维与安全中心 |

## 4. 各区域详细设计

### 4.1 页面标题区

| 属性 | 规格 |
|------|------|
| 标题 | `text-display` 28px / 700 / `--neutral-900` |
| 副标题 | `text-caption` 12px / 400 / `--neutral-500`，margin-top: 4px |
| 右侧按钮 | "导出配置"(Secondary) + "变更记录"(Ghost，带 HistoryOutlined 图标) |
| 间距 | 底部 margin-bottom: 24px |

### 4.2 分组导航（左侧）

| 属性 | 规格 |
|------|------|
| 容器 | width: 180px / position: sticky / top: 80px(顶栏56px+padding24px) |
| 导航项 | padding: 10px 16px / font-size: 14px / 400 / border-radius: 6px |
| 默认态 | 文字 `--neutral-600` / 背景 transparent |
| 活跃态 | 文字 `--semantic-600` / 背景 `--semantic-50` / font-weight: 500 / 左侧3px蓝色竖条 |
| hover 态 | 背景 `--neutral-100` |
| 项间距 | gap: 4px |
| 活跃检测 | 基于 IntersectionObserver 检测右侧滚动位置，自动高亮对应导航项 |

### 4.3 配置组卡片（通用结构）

| 属性 | 规格 |
|------|------|
| 容器 | 背景 `--neutral-0` / border: 1px solid `--neutral-200` / border-radius: 12px / padding: 24px |
| 组标题 | `text-h2` 18px / 600 / `--neutral-900` / margin-bottom: 4px |
| 组描述 | 14px / 400 / `--neutral-500` / margin-bottom: 20px |
| 表单布局 | 垂直布局 / label 在上方 / gap: 16px |
| label | 14px / 500 / `--neutral-700` / margin-bottom: 6px |
| 底部操作栏 | margin-top: 20px / padding-top: 16px / border-top: 1px solid `--neutral-100` / flex justify-end / gap: 12px |
| 恢复默认按钮 | Ghost 类型 / 13px / 需二次确认 |
| 保存按钮 | Primary 类型 / 14px / 无变更时 disabled |
| 保存成功反馈 | `a-message.success("保存成功")` |

### 4.4 基础配置组

### 4.9 配置变更记录（抽屉）

| 属性 | 规格 |

## 5. 数据展示规范

| 场景 | 规范 |
|------|------|
| API Key 脱敏 | 显示前4位 + **** + 后4位，如 "sk-a****fg2h" |
| 密码字段 | 一律显示 "••••••••"，点击眼睛图标后显示原文(3秒后自动隐藏) |
| 数字输入 | 带单位后缀文字（分钟、MB等） |
| Temperature | 保留1位小数，如 "0.7" |
| 文件大小限制 | 显示为 "50 MB"，输入时纯数字 |
| 加载态 | 每个配置组独立 skeleton / 表单字段区域显示条形骨架 |
| 保存中 | 按钮 loading 态 + 文字变为 "保存中..." |
| 保存成功 | `a-message.success` 全局提示 / 持续 2s |
| 保存失败 | `a-message.error` 显示具体错误 / 表单字段红色标注 |
| 未修改状态 | 保存按钮 disabled / opacity: 0.5 |
| 测试连通性结果 | 按钮下方内联展示：绿色对勾 "连接成功" 或 红色叉 "连接失败: 超时" |
| 空配置 | 输入框显示 placeholder 提示文字 |

## 6. 响应式适配

| 断点 | 布局变化 |
|------|----------|
| Desktop L (≥1920px) | 左侧导航180px + 右侧配置区 / 表单双列布局(短字段并排) / 页面 padding: 32px |
| Desktop M (1440-1919px) | 左侧导航180px + 右侧配置区 / 表单单列布局 / 页面 padding: 24px |
| Desktop S (1280-1439px) | 左侧导航隐藏，改为顶部横向 Tab 切换 / 表单单列 / 页面 padding: 20px |
| Tablet (<1280px) | 顶部横向 Tab(可滚动) / 表单全宽单列 / 配置组卡片 padding 缩减为 16px / 页面 padding: 16px |

表单字段响应式：
- ≥1920px：短字段(数字、Select)可并排2列(grid-template-columns: 1fr 1fr)
- <1920px：所有字段单列堆叠

### 4.10 左侧导航折叠规则

| 条件 | 行为 |
|------|------|
| ≥1280px | 左侧 sticky 导航固定显示 |
| <1280px | 导航消失，改为配置区上方的水平 Segmented 组件 |
| 滚动时 | IntersectionObserver 检测各组标题进入视口，自动更新导航高亮 |
| 点击导航 | scrollIntoView({ behavior: 'smooth', block: 'start' }) |
