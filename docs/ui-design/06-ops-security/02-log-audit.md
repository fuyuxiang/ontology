# 日志与审计（Log & Audit）

> 运维与安全中心 · 全链路日志查询与操作审计追溯

---

## 1. 页面概述

| 维度 | 说明 |
|------|------|
| 页面定位 | 平台日志统一查询入口与操作审计中心，支持实时尾部追踪和历史回溯 |
| 目标用户 | 系统管理员、运维工程师、安全审计员 |
| 核心价值 | 快速定位系统故障根因，完整记录用户操作行为，满足合规审计要求 |
| 路由路径 | `/ops/log-audit` |
| 所属分组 | 运维与安全中心 |

---

## 2. 功能清单

| # | 功能点 | 优先级 | 交互类型 | 说明 |

## 4. 各区域详细设计

### 4.1 页面标题区

| 属性 | 规格 |
|------|------|
| 标题 | `text-display` 28px / 700 / `--neutral-900` |
| 副标题 | `text-caption` 12px / 400 / `--neutral-500`，margin-top: 4px |
| 导出按钮组 | 右对齐，Secondary 类型，带下拉箭头图标 |
| 间距 | 底部 margin-bottom: 16px |

### 4.2 Segmented 切换

| 属性 | 规格 |
|------|------|
| 组件 | `a-segmented`（Ant Design Vue） |
| 选项 | "日志查询" / "审计记录" |
| 尺寸 | 默认尺寸，宽度 auto |
| 样式 | 与 AuditLogView 中现有实现一致 |

### 4.3 日志筛选工具栏

| 属性 | 规格 |
|------|------|
| 容器 | flex wrap / gap: 12px / align-items: center |
| 时间选择器 | `a-range-picker`，width: 280px / 预设 ranges 快捷按钮 |
| 级别过滤 | `a-checkbox-group`，行内标签样式，每个标签带对应颜色圆点 |
| 级别颜色 | DEBUG: `--neutral-500` / INFO: `--status-info` (#339af0) / WARN: `--status-warning` (#f59f00) / ERROR: `--status-error` (#fa5252) |
| 服务筛选 | `a-select`，width: 160px，placeholder "全部服务" |
| 关键词输入 | `a-input-search`，flex: 1，min-width: 200px |
| 实时流开关 | `a-switch` + 标签文字 "实时"，右侧对齐 |

### 4.4 日志统计条

| 属性 | 规格 |
|------|------|
| 容器 | 背景 `--neutral-50` / padding: 8px 16px / border-radius: 6px |
| 文字 | 13px / 400 / `--neutral-600` |
| 数量 | font-weight: 600 / 各级别对应颜色 |
| 分隔 | 使用竖线 "│" 分隔，`--neutral-200` 颜色 |

### 4.5 日志列表

| 属性 | 规格 |
|------|------|
| 容器 | 虚拟滚动列表，行高估算 56px（折叠）/ 动态（展开） |
| 日志行（折叠） | padding: 12px 16px / border-bottom: 1px solid `--neutral-100` |
| 时间戳 | 13px / font-mono / `--neutral-500` / width: 160px |
| 级别标签 | 11px / padding: 2px 8px / border-radius: 4px / 对应级别配色（浅底深字） |
| 服务名 | 13px / 500 / `--neutral-700` / width: 120px |
| 消息内容 | 14px / 400 / `--neutral-800` / flex: 1 / 单行截断 ellipsis |
| 日志行（展开） | 额外内容区 padding: 12px 16px 12px 176px（缩进对齐） |
| 堆栈信息 | `text-code` 13px / font-mono / 背景 `--neutral-50` / padding: 12px / border-radius: 6px / max-height: 300px / overflow: auto |
| hover 行 | 背景 `--neutral-50` / cursor: pointer |
| 实时模式指示 | 列表顶部绿色脉冲条 + "实时接收中..." 文字 |

### 4.6 审计记录表格

| 属性 | 规格 |
|------|------|
| 组件 | 自定义 table（与 AuditLogView 风格统一） |
| 表头 | 背景 `--neutral-50` / 12px / 600 / `--neutral-500` |
| 行高 | 48px |
| 时间列 | width: 160px / font-mono / 13px |
| 操作人列 | width: 100px / 14px / 500 |
| 操作类型列 | width: 100px / 标签样式（与 AuditLogView actionMeta 配色一致） |
| 操作类型配色 | 创建: #10b981 / 修改: #5c7cfa / 删除: #fa5252 / 登录: #339af0 / 导出: #f59f00 |
| 目标对象列 | flex: 1 / 14px / 可点击跳转（蓝色链接样式） |
| IP地址列 | width: 120px / font-mono / 13px / `--neutral-500` |
| 结果列 | width: 80px / 居中 / 成功: 绿色对勾图标 / 失败: 红色叉图标 |
| hover 行 | 背景 `--neutral-50` / cursor: pointer |
| 点击行为 | 点击行打开右侧审计详情抽屉 |

### 4.7 审计详情抽屉

| 属性 | 规格 |
|------|------|
| 组件 | `a-drawer`，width: 480px，从右侧滑入 |
| 头部 | 标题 "操作详情" / 18px / 600 / 关闭按钮 |
| 元信息区 | 背景 `--neutral-50` / padding: 16px / border-radius: 8px / 键值对布局 |
| 元信息内容 | 操作人、时间、IP地址、User-Agent、操作类型、结果状态 |
| 变更对比区 | 标题 "字段变更" / 表格：字段名 + 变更前(红色删除线) + 变更后(绿色) |
| 无变更 | 显示 "此操作无字段变更记录" / 居中 / `--neutral-400` |

### 4.8 导出功能

| 属性 | 规格 |
|------|------|
| 触发 | 页面标题右侧按钮 |
| 下拉选项 | "导出 CSV" / "导出 Excel" |
| 行为 | 导出当前筛选条件下的全部数据（需后端分页拉取） |
| 加载态 | 按钮变为 loading 状态 + 文字 "导出中..." |
| 完成 | `a-message` 成功提示 + 浏览器自动下载 |
| 数据量限制 | 超过 10,000 条时弹窗确认 "数据量较大，导出可能需要较长时间，是否继续？" |

## 5. 数据展示规范

| 场景 | 规范 |
|------|------|
| 时间戳格式 | 日志: "YYYY-MM-DD HH:mm:ss.SSS" / 审计: "YYYY-MM-DD HH:mm:ss" |
| 日志总数 | 千位分隔符，如 "12,458 条" |
| IP 地址 | 原样展示，font-mono |
| 加载态 | 日志列表: 3行骨架屏(shimmer) / 审计表格: 行级骨架屏 |
| 空状态(日志) | 图标: 搜索放大镜 / 标题: "未找到匹配日志" / 描述: "尝试调整时间范围或筛选条件" |
| 空状态(审计) | 图标: 文件图标 / 标题: "暂无审计记录" / 描述: "系统操作将自动记录在此" |
| 错误态 | 红色 Alert 组件，显示错误消息 + "重试"按钮 |
| 实时流断开 | 黄色警告条: "实时连接已断开，正在重连..." + 自动重连逻辑 |
| 长文本 | 日志消息超过单行时截断省略号，展开后完整显示 |

## 6. 响应式适配

| 断点 | 布局变化 |
|------|----------|
| Desktop L (≥1920px) | 工具栏单行展示 / 表格全列可见 / 页面 padding: 32px |
| Desktop M (1440-1919px) | 工具栏单行展示 / 表格全列可见 / 页面 padding: 24px |
| Desktop S (1280-1439px) | 工具栏可换行(flex-wrap) / 隐藏IP列 / 页面 padding: 20px |
| Tablet (<1280px) | 工具栏两行 / 隐藏IP列和User-Agent / 审计抽屉改为全屏Modal / 页面 padding: 16px |

日志列表响应式：
- ≥1440px：时间戳 + 级别 + 服务名 + 消息 同行展示
- <1440px：时间戳和服务名第一行，消息内容第二行
