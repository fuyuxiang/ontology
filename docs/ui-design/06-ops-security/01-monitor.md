# 运维监控（Monitor）

> 运维与安全中心 · 系统健康状况实时感知与预警

---

## 1. 页面概述

| 维度 | 说明 |
|------|------|
| 页面定位 | 系统运行状态实时监控大盘，包含资源、服务、安全、链路追踪和容灾五大维度 |
| 目标用户 | 系统管理员、运维工程师、安全管理员 |
| 核心价值 | 实时掌握平台健康状态，快速定位性能瓶颈与安全威胁，保障系统稳定运行 |
| 路由路径 | `/ops/monitor` |
| 所属分组 | 运维与安全中心 |

---

## 2. 功能清单

| # | 功能点 | 优先级 | 交互类型 | 说明 |

## 4. 各区域详细设计

### 4.1 页面标题区

| 属性 | 规格 |
|------|------|
| 标题 | `text-display` 28px / 700 / `--neutral-900` |
| 副标题 | `text-caption` 12px / 400 / `--neutral-500`，margin-top: 4px |
| 自动刷新控件 | 右对齐，包含开关(Switch) + 间隔下拉(Select)，高度 32px |
| 区域底部间距 | 16px |

### 4.2 Tab 栏

| 属性 | 规格 |
|------|------|
| 组件 | 自定义按钮组（非 Ant Tabs，与现有实现一致） |
| 按钮样式 | 14px / 500 / padding: 8px 16px / border-radius: 6px |
| 默认态 | 背景 transparent / 文字 `--neutral-600` |
| 活跃态 | 背景 `--semantic-50` / 文字 `--semantic-600` / font-weight: 600 |
| hover 态 | 背景 `--neutral-100` |
| 间距 | 按钮间 gap: 4px |

### 4.3 资源仪表盘卡片

| 属性 | 规格 |
|------|------|
| 容器 | 3列等宽网格，gap: 16px |
| 卡片 | 背景 `--neutral-0` / border: 1px solid `--neutral-100` / border-radius: 12px / padding: 20px |
| 标题 | `text-body` 14px / 600 / `--neutral-700`，居中 |
| SVG 仪表盘 | viewBox: 0 0 100 50 / 半圆弧 / stroke-width: 8 / stroke-linecap: round |

### 4.8 容灾状态面板

| 属性 | 规格 |
|------|------|
| 备份策略卡片 | 网格2列 / 卡片样式 / 显示：策略名称、频率、保留时间、最近执行状态 |

| 状态指示 | 正常同步: 绿色脉冲动画 / 延迟过大: 黄色警告 / 断开: 红色 |
| 手动切换按钮 | Danger 类型按钮 / 二次确认 Modal |
| 备份历史 | 表格：时间、类型(全量/增量)、大小、耗时、状态 |

## 5. 数据展示规范

| 场景 | 规范 |
|------|------|
| CPU/内存/磁盘百分比 | 保留1位小数，如 "45.2%" |
| 存储容量 | GB 为单位，保留1位小数，如 "8.2GB / 12GB" |
| 响应时间 | 毫秒为单位，整数，如 "320ms" |
| 运行时间 | 自动换算，如 "45天 12小时" |
| 时间戳 | 格式 "YYYY-MM-DD HH:mm:ss"，去掉T分隔符 |
| 加载态 | 仪表盘区域显示 Skeleton 占位 / 表格显示行级骨架屏 |
| 空状态 | 居中图标(64px) + 标题(16px) + 描述(14px) |
| 错误态 | 红色边框卡片 + 错误图标 + 描述 + "重试"按钮 |
| 无数据 | 仪表盘显示 "--" / 数字显示 "0" / 列表显示空状态 |

## 6. 响应式适配

| 断点 | 布局变化 |
|------|----------|
| Desktop L (≥1920px) | 资源卡片3列 / 系统信息3列 / 页面 padding: 32px |
| Desktop M (1440-1919px) | 资源卡片3列 / 系统信息3列 / 页面 padding: 24px |
| Desktop S (1280-1439px) | 资源卡片3列 / 系统信息2列 / 页面 padding: 20px |
| Tablet (<1280px) | 资源卡片纵向堆叠 / 系统信息2列 / Tab栏可横向滚动 / 页面 padding: 16px |

链路追踪瀑布图在窄屏下：
- ≥1440px：Span标签与瀑布条同行显示
- <1440px：Span标签独占一行，瀑布条另起一行
