# 数据接入（Connections）— UI 设计文档

---

## 1. 页面概述

| 维度 | 说明 |
|------|------|
| 页面定位 | 数据集成模块的入口页面，统一管理所有外部数据源连接的生命周期 |
| 用户角色 | 数据工程师、平台管理员 |
| 核心价值 | 一站式创建、测试、监控数据源连接，保障下游资产接入的稳定性 |
| 路由路径 | `/data-integration/connections` |

---

## 4. 各区域详细设计

### 4.1 页面标题区

| 属性 | 值 |
|------|-----|
| 标题 | "数据接入" |
| 字体 | 22px / 600 / `--neutral-900` |
| 副标题 | "统一管理数据源连接、凭据与访问权限" |
| 副标题字体 | 12px / 400 / `--neutral-500` |
| 内边距 | `24px 32px 0` |
| 底部间距 | `--space-5`(20px) |

### 4.2 统计卡片栏

- **布局**: `display: flex; align-items: center; gap: 32px`
- **容器**: 白色背景、`border-radius: 8px`、阴影 sm、内边距 16px
- **每个统计项**: 使用 `<a-statistic>` 组件
  - 标题: 12px / 400 / `--neutral-500`
  - 数值: 28px / 700 / 对应状态色
  - 图标前缀: 24px、对应状态色
- **在线数值色**: `#10b981`（成功绿）
- **离线数值色**: `#6b7280`（中性灰）
- **异常数值色**: `#ef4444`（错误红）
- **右侧按钮**: Primary 类型，"新增数据源"，高 36px

### 4.3 工具栏

| 属性 | 值 |
|------|-----|
| 布局 | `display: flex; gap: 12px; align-items: center` |
| 背景 | `#ffffff` |
| 圆角 | 8px |
| 内边距 | `12px 16px` |
| 搜索框宽度 | 300px |
| 分段控件 | `<a-segmented>` 分类过滤 |
| 视图切换 | `<a-segmented>` 图标模式（AppstoreOutlined / UnorderedListOutlined） |

**交互行为**:
- 搜索输入即时过滤（debounce 300ms）
- 分类切换重新请求列表接口
- 视图切换保存到 localStorage

### 4.4 卡片视图

- **网格**: `<a-row :gutter="[16, 16]">`
- **列宽**: xs=24, sm=12, lg=8, xl=6
- **卡片组件**: `<a-card hoverable>`
- **卡片结构**:

```
┌─────────────────────────────┐
│  [数据源类型图标 40×40]       │
│  名称 (14px/600/#111827)     │
│  分类标签 + 类型标签          │
│                              │
│  ● 在线  /  ○ 离线           │  ← 状态指示器
│  最近同步: 2024-01-15 10:30  │  ← 12px/--neutral-500
│                              │
│  ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─  │  ← 虚线分割
│  [测试] [编辑] [删除]        │  ← hover 时显示
└─────────────────────────────┘
```

| 子元素 | 规格 |
|--------|------|
| 图标区 | 40×40px, border-radius 10px, 品牌色背景 |
| 名称 | 14px / 600 / `--neutral-900`，单行省略 |
| 分类标签 | `<a-tag>` 12px，颜色映射见下 |
| 状态点 | 8×8px 圆点 + 文字（在线=#10b981, 离线=#6b7280, 异常=#ef4444） |
| 同步时间 | 12px / `--neutral-500`，使用相对时间（dayjs.fromNow） |
| 操作按钮 | type="link" size="small"，初始 opacity:0，hover 卡片时 opacity:1 |

**分类颜色映射**:
- 数据库: `#3b82f6`
- 对象存储: `#8b5cf6`
- 文件传输: `#f59e0b`
- 消息队列: `#10b981`
- API: `#ec4899`

**Hover 效果**:
- `border-color: --semantic-300`
- `box-shadow: 0 4px 12px rgba(59,130,246,0.08)`
- `transform: translateY(-1px)`
- 过渡: 150ms ease

### 4.5 列表视图

使用 `<a-table size="middle">` 组件:

| 列名 | 宽度 | 内容 |
|------|------|------|
| 名称 | 180px | 可点击链接样式 |
| 分类 | 100px | `<a-tag>` 带颜色 |
| 类型 | 120px | `<a-tag>` 显示具体驱动 |
| 端点 | auto | `<code>` 等宽字体，host:port 格式 |
| 状态 | 100px | 圆点+文字 |
| 读写 | 80px | `<a-tag>` 可写=orange / 只读=default |
| 操作 | 200px | 测试/浏览/编辑/删除 |

- 行高: 48px
- 分页: `pageSize: 20`，右对齐

### 4.6 新建连接向导

**第一步：选择数据源类型（模态框）**

- 宽度: 720px
- 内容: 3列网格，每格为可点击卡片
- 每张卡片: 图标(44×44) + 类型名(14px/600) + 支持的驱动列表(11px) + 提示文字(11px)
- hover: `border-color: #3b82f6`, `transform: translateY(-1px)`
- 不可用: `opacity: 0.45`, `cursor: not-allowed`

**第二步：填写参数（抽屉表单）**

- 抽屉宽度: 560px
- 表单布局: `layout="vertical"`
- 表单项:
  - 连接名称 *（必填）
  - 数据库类型 *（Select）
  - 主机地址 *（Input）
  - 端口 *（InputNumber，预填默认端口）
  - 数据库名 *（Input）
  - 用户名 *（Input）
  - 密码 *（Input.Password）
  - 读写模式（Switch）
  - 备注（Textarea）
- 底部固定操作栏: [取消] [测试连接] [保存]

**测试连接反馈**:
- 测试中: 按钮 loading + 旋转图标
- 成功: `<a-alert type="success">` "连接成功，延迟 23ms"
- 失败: `<a-alert type="error">` 显示错误信息

### 4.7 删除确认弹窗

- 组件: `<a-modal>`
- 检测关联资产数:
  - 无关联: 提示"可直接删除"，绿色文字
  - 有关联: 红色警告 + 级联删除复选框
- 按钮: 禁用状态（有资产且未勾选级联时）

## 5. 数据展示规范

| 场景 | 规范 |
|------|------|
| 数字格式化 | 千分位分隔：`toLocaleString()` |
| 时间显示 | 列表中使用相对时间（5分钟前）；详情中使用完整格式 YYYY-MM-DD HH:mm:ss |
| 空状态 | 居中插图(64×64) + 标题(16px/600) + 描述(14px/400) + 操作按钮 |
| 加载态 | 表格: `<a-table :loading="true">`; 卡片: 骨架屏 `<a-skeleton>` |
| 错误态 | `<a-result status="error">` + 重试按钮 |
| 状态刷新 | 测试中显示 `<SyncOutlined spin>` + 橙色 |
