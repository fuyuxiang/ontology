# 资产目录（Assets）— UI 设计文档

---

## 1. 页面概述

| 维度 | 说明 |
|------|------|
| 页面定位 | 数据集成模块的资产管理中心，以树形结构组织并展示所有已接入的数据资产 |
| 用户角色 | 数据工程师、数据分析师、本体建模人员 |
| 核心价值 | 提供统一的数据资产视图，支持快速定位表/字段信息，为本体映射奠定基础 |
| 路由路径 | `/data-integration/assets` |

## 4. 各区域详细设计

### 4.1 页面标题区

| 属性 | 值 |
|------|-----|
| 标题 | "资产目录" |
| 字体 | 22px / 600 / `--neutral-900` |
| 副标题 | "统一管理可用数据资产" |
| 副标题字体 | 12px / 400 / `--neutral-500` |
| 内边距 | `24px 32px 0` |
| 底部间距 | `--space-5`(20px) |

### 4.2 统计栏

- **组件**: `<a-statistic>`
- **布局**: `display: flex; gap: 32px; align-items: center`
- **容器**: 白色背景、圆角 8px、阴影 xs、内边距 16px
- 资产总数: 数值色 `--neutral-900`
- 结构化: 数值色 `#3b82f6`
- 非结构化: 数值色 `#f59e0b`

### 4.3 分类 Tabs

- **组件**: `<a-tabs>`
- **Tab 项**: 全部 / 结构化 / 非结构化
- **样式**: 默认线条风格，活跃态下划线 `--semantic-500`
- **切换行为**: 过滤列表 + 更新统计数字

### 4.4 工具栏

| 元素 | 组件 | 规格 |
|------|------|------|
| 搜索框 | `<a-input-search>` | 宽 340px，allow-clear |
| Domain 过滤 | `<a-select>` | 宽 200px，allow-clear |
| 视图切换 | `<a-segmented>` | grid / table |

- 容器: 白色背景、圆角 8px、内边距 `12px 16px`、底部间距 12px

### 4.5 左侧树形导航

| 属性 | 值 |
|------|-----|
| 宽度 | 260px（可拖拽调整，min 200px, max 400px） |
| 背景 | `#ffffff` |
| 右边框 | `1px solid --neutral-200` |
| 内边距 | `12px 0` |
| 组件 | `<a-tree>` |

**树节点层级**:

| 层级 | 图标 | 文字 | 说明 |
|------|------|------|------|
| L1 数据源 | DatabaseOutlined (20px) | 14px/600/`--neutral-800` | 展开箭头 |
| L2 数据库 | FolderOutlined (16px) | 13px/400/`--neutral-700` | |
| L3 Schema | FolderOpenOutlined (16px) | 13px/400/`--neutral-600` | 可选层（部分数据库无 Schema） |
| L4 表/视图 | TableOutlined (16px) | 13px/400/`--neutral-700` | 叶节点 |

**交互行为**:
- 点击叶节点: 右侧展示表详情
- 右键菜单: 同步 Schema / Profile / 删除
- 选中态: 背景 `--semantic-50`，左侧 3px 蓝色竖条
- hover: 背景 `#f0f4ff`
- 展开/折叠: 250ms ease-out 动画
- 搜索匹配: 高亮命中文字（背景 `#fef9c3`）

**顶部搜索框**:
- 树内局部搜索，位于树区域顶部
- 宽度 100%，高度 32px，圆角 6px
- 搜索时自动展开匹配路径

### 4.6 右侧内容区 — 表详情面板

当选中某张表时，右侧展示详情:

**头部信息卡片**:
```
┌──────────────────────────────────────────────────────────┐
│  表名: customer_orders                    [同步][Profile] │
│  连接: MySQL生产库 · Schema: public                       │
│  标签: [核心表] [交易域] [+添加]                          │
│  行数: 1,234,567  字段数: 24  最后同步: 2024-01-15       │
└──────────────────────────────────────────────────────────┘
```

| 属性 | 规格 |
|------|------|
| 表名 | 18px / 600 / `--neutral-900` |
| 连接信息 | 12px / 400 / `--neutral-500` |
| 标签 | `<a-tag>` 可编辑，点击"+"弹出标签选择器 |
| 统计数据 | 14px / `--neutral-700`，数值 font-weight: 600 |
| 按钮组 | 右上角，Secondary 样式 |

**Tab 切换区**: `<a-tabs>` 包含 3 个面板

#### Tab 1: 字段列表

- **组件**: `<a-table size="small">`
- **列定义**:

| 列名 | 宽度 | 内容 |
|------|------|------|
| 字段名 | 160px | 等宽字体 13px |
| 数据类型 | 120px | `<a-tag color="blue">` |
| 可空 | 60px | ✓ / ✗ 图标 |
| 主键 | 60px | 🔑 图标（如是） |
| 注释 | auto | 14px，支持行内编辑 |
| 标签 | 150px | 标签组，可添加 |
| 空值率 | 80px | 百分比 + 微型进度条 |
| 基数 | 80px | 数字，千分位 |

- 行高: 40px
- hover: 行背景 `--neutral-50`
- 字段名字体: `--font-mono`, 13px

#### Tab 2: 统计信息

- **布局**: 2列网格 `grid-template-columns: 1fr 1fr`
- **内容**:
  - 行数/列数/存储大小/最后更新时间
  - 字段类型分布（饼图，使用 AntV G2）
  - TOP10 高空值率字段（条形图）
  - 索引信息列表

#### Tab 3: 样本数据

- **组件**: `<a-table>` 动态列
- **数据**: 前 100 行原始数据
- **特性**:
  - 水平滚动（字段多时）
  - 列头可排序
  - 单元格超长内容 tooltip 显示
  - 操作栏: [刷新样本] [复制为 CSV]

### 4.7 卡片网格视图（未选中表时）

- **网格**: `<a-row :gutter="[16,16]">`
- **列宽**: xs=24, sm=12, lg=8, xl=6
- **卡片内容**:
  - 名称(14px/600) + 类型标签 + 别名标签
  - 状态标签
  - 描述(12px, 2行省略)
  - 标签组(最多4个)
  - 底部: 记录数 / 刷新策略 / 连接名

**hover 效果**: 同设计系统卡片规范（translateY(-1px) + shadow md）

### 4.8 新建资产抽屉

- **宽度**: 560px
- **两种模式**:
  - 结构化资产: 选择连接 → 选择表 → 填写别名/描述/标签/刷新策略
  - 非结构化资产: 选择连接 → 填写名称/路径/文件类型/描述
- **表单组件**: Ant Design Vue Form，layout="vertical"
- **底部操作栏**: 固定定位，[取消] [保存]

## 5. 数据展示规范

| 场景 | 规范 |
|------|------|
| 行数格式化 | `toLocaleString()`，如 1,234,567 |
| 空值率 | 百分比保留 1 位小数 + 微型条形图(高4px, 宽40px) |
| 时间显示 | 列表用相对时间；详情用 YYYY-MM-DD HH:mm |
| 空状态 | 未选中表: 居中引导文案"从左侧树中选择一张表查看详情" |
| 加载态 | 树: `<a-tree>` 内置 loading 节点; 详情: `<a-skeleton active>` |
| 错误态 | `<a-alert type="error">` 内嵌在对应区域 |
| 无数据 | `<a-empty>` + 引导按钮"点击「新建资产」开始登记" |
| 搜索无结果 | `<a-empty description="未找到匹配的资产">` |

## 6. 响应式适配

| 断点 | 树导航 | 内容区 | 卡片列数 |
|------|--------|--------|---------|
| ≥1920px | 260px 固定 | 自适应 | 4列 |
| 1440-1919px | 260px 固定 | 自适应 | 3列 |
| 1280-1439px | 200px 收窄 | 自适应 | 2列 |
| <1280px | 折叠为抽屉模式(点击汉堡按钮弹出) | 全宽 | 1列 |

**树导航折叠逻辑**:
- <1280px 时树区域隐藏，顶部出现展开按钮
- 点击按钮从左侧滑出抽屉(宽 280px)
- 选中节点后自动关闭抽屉

**详情面板适配**:
- <1440px 时 Tab 内表格开启水平滚动
- <1280px 时统计信息从 2列 变为 1列
