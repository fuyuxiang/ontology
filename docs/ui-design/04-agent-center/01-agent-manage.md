# 智能体管理 — UI 设计文档

> 所属分组：智能体应用中心  
> 版本：v1.0  
> 设计系统参考：`00-design-system.md`

---

## 1. 页面概述

| 维度 | 说明 |
|------|------|
| 页面定位 | 智能体的全生命周期管理入口，提供列表浏览与画布式可视化编辑 |
| 用户角色 | 本体工程师、AI 应用开发者 |
| 核心价值 | 以零代码/低代码方式组装 Agent 流程，实现知识推理与服务编排的可视化 |

页面分为两个层级：
- **列表页**：展示所有 Agent 的卡片概览，支持搜索/筛选/创建
- **编辑页**：基于 VueFlow 的画布式节点编排器

## 4. 各区域详细设计

### 4.1 列表页 — 页面标题区

| 属性 | 规格 |
|------|------|
| 标题 | "智能体管理"，`--text-h1` 22px/600/`--neutral-900` |
| 描述 | 14px/400/`--neutral-500`，距标题下方 4px |
| 新建按钮 | Primary Button，高度 36px，图标 + 文字"新建智能体" |
| 间距 | 标题区与工具栏间距 `--space-5`(20px) |

### 4.2 列表页 — 工具栏

| 组件 | 规格 |
|------|------|
| 搜索框 | `a-input-search`，宽度 280px，placeholder "搜索智能体名称..." |
| 状态筛选 | `a-select`，宽度 120px，选项：全部/草稿/已发布 |
| 排序 | `a-select`，宽度 140px，选项：最近创建/最近更新/名称 |
| 间距 | 各组件间距 `--space-3`(12px) |
| 对齐 | 搜索左对齐，筛选和排序右对齐 (flex justify-between) |

### 4.3 列表页 — Agent 卡片

| 属性 | 规格 |
|------|------|
| 容器 | 网格布局 `grid-template-columns: repeat(auto-fill, minmax(280px, 1fr))`，gap 16px |
| 卡片 | 白色背景，border 1px `--neutral-200`，圆角 8px，padding 20px |
| 卡片hover | translateY(-2px) + shadow-md，过渡 200ms ease |
| Agent图标 | 40×40px 圆角方形(8px)，背景渐变(基于Agent类型)，居中显示机器人图标 |
| 名称 | `--text-h3` 15px/600/`--neutral-900`，单行省略 |
| 状态标签 | 草稿: 背景`#fff9db`文字`#f59f00` / 已发布: 背景`#e6fcf5`文字`#12b886` |
| 描述 | 14px/400/`--neutral-500`，2行省略(line-clamp: 2) |
| 底部 | flex, 左侧"创建于 2024-01-01" caption 12px，右侧操作按钮(更多...) |
| 操作菜单 | `a-dropdown`：编辑/复制/发布/删除(红色) |

### 4.4 编辑页 — 顶栏

| 属性 | 规格 |
|------|------|
| 高度 | 48px |
| 背景 | `#ffffff`，底部 border 1px `--neutral-200` |
| 返回按钮 | Ghost Button，`← 返回`，点击回到列表页 |
| 分隔竖线 | 1px × 20px，颜色 `--neutral-200`，margin 0 12px |
| Agent名称 | 15px/600，可编辑态(hover 显示编辑图标) |
| 状态标签 | 同卡片状态标签规格 |

| 操作按钮组 | 保存(Secondary) + 发布(Primary Outline) + 设置(Ghost) |

### 4.5 编辑页 — 节点库面板

| 属性 | 规格 |
|------|------|
| 宽度 | 220px，固定 |
| 背景 | `--neutral-0` (#ffffff) |
| 右边框 | 1px solid `--neutral-200` |
| 搜索框 | 宽度 100%，高度 32px(小尺寸)，margin-bottom 12px |
| 分组标题 | 11px/600/`--neutral-500`，text-transform: uppercase，margin 12px 0 6px |
| 节点项 | 高度 32px，padding 0 12px，圆角 6px，flex(图标+文字+添加图标) |
| 节点项hover | 背景 `--neutral-50`，右侧出现 + 图标 |
| 节点图标 | 16px，各类别不同颜色(输入#339af0/处理#7950f2/输出#20c997/条件#f59f00) |
| 拖拽态 | cursor: grab → grabbing，节点项半透明0.5 |

**节点分组**：

| 分组 | 包含节点 |
|------|----------|
| 输入 | 文本输入、变量注入、文件上传 |
| 处理 | LLM调用、规则执行、本体查询、API调用、数据写入、代码执行 |
| 输出 | 文本响应、JSON输出、文件输出 |
| 条件 | 条件分支、循环、等待 |

### 4.6 编辑页 — 画布区

| 属性 | 规格 |
|------|------|
| 背景 | `#f8fafc`(亮色) / `#1a1b1e`(暗色) |

| 节点对齐 | snap-to-grid [16, 16] |

### 4.7 编辑页 — 配置面板

| 属性 | 规格 |
|------|------|
| 宽度 | 280px |

| 重试次数 | `a-slider` + 数字 | 0-5次 |
| 重试间隔 | `a-select` | 1s/2s/5s/10s |

## 5. 数据展示规范

### 5.1 数字格式化

| 数据类型 | 格式 | 示例 |
|----------|------|------|
| 创建时间 | YYYY-MM-DD HH:mm | 2024-06-01 14:30 |
| 相对时间 | "x分钟前/x小时前/x天前" | 3小时前 |
| 节点数量 | 整数 | 12 个节点 |
| 超时时间 | 带单位 | 30s |

### 5.2 空状态

| 场景 | 图标 | 标题 | 描述 | 操作 |
|------|------|------|------|------|
| 列表无数据 | 机器人插图 120×120 | 暂无智能体 | 创建第一个智能体开始智能服务编排 | [新建智能体] |
| 搜索无结果 | 搜索图标 64×64 | 未找到匹配项 | 尝试更换关键词或清除筛选条件 | [清除筛选] |
| 画布空白 | 拖拽引导插图 | 从左侧节点库拖入节点 | 或点击节点旁的 + 按钮添加到画布 | — |

### 5.3 加载态

| 场景 | 方案 |
|------|------|
| 列表加载 | 骨架屏 (a-skeleton)，3行×3列卡片骨架 |
| 画布加载 | 中央 Spin + "加载中..." |
| 保存中 | 按钮 loading 态 + 文字"保存中..." |
| 发布中 | 按钮 loading 态 + 文字"发布中..." |

### 5.4 错误态

| 场景 | 方案 |
|------|------|
| 列表加载失败 | `a-result` status="error" + 重试按钮 |
| 保存失败 | `a-message.error` 顶部提示 |
| 节点配置错误 | 节点边框变红 + 配置面板显示错误提示 |
